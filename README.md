## Graph text format
    N M
    s_1 t_1 (w_1)
    ...
    s_i t_i (w_i)
    ...
    s_M t_M (w_M)

* N: number of nodes
* M: number of edges
* s_i: source of i-th edge
* t_i: destination of i-th edge
* w_i: weight of i-th edge
  + w_i is optional. If not specified, default value of 1 is used.
